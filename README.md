# Software Developer Portfolio

A responsive, single-page portfolio website built with plain HTML, CSS, and JavaScript.

## Features

- Clean dark visual style
- Fully responsive desktop, tablet, and mobile layout
- Mobile navigation menu
- Scroll reveal animations
- Active navigation section indicator
- About, skills, projects, process, and contact sections
- No framework or build process required

## Files

- `index.html` — page structure and content
- `styles.css` — complete visual design and responsive layout
- `script.js` — navigation and animation behavior

## Run locally

Open `index.html` directly in your browser.

For the best local development experience, use the Live Server extension in Visual Studio Code.

## Customize

Search inside `index.html` and replace:

- `Daner`
- `your-email@example.com`
- `github.com/yourusername`
- `linkedin.com/in/yourusername`
- Project names and descriptions
- Skills and technologies
